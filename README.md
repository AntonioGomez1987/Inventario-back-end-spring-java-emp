# Sistema Inventario Empanadas

### Documentación del proyecto
Es un proyecto realizado con java, spring boot, con dependencias como Spring Web para la utilizacion de un servidor
Tomcat que incluyen peticiones de tipo RESTful, Spring Data JPA para la comunicacion con la 
DB, MySQL Driver este proyecto lo trabajo con la DB de MySQL,
Lombok para la creacion de las clases de entida.
